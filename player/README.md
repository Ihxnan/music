# 音乐播放器 (Music Player)

一个现代化的网页音乐播放器，支持播放、搜索、下载等功能。

## 功能特性

- 🎵 **音乐库浏览** - 按艺术家分类浏览所有音乐
- 🔍 **智能搜索** - 支持按歌曲名和艺术家搜索
- ▶️ **完整播放控制** - 播放、暂停、上一首、下一首
- 📊 **进度控制** - 可拖动的进度条
- 🔊 **音量控制** - 平滑的音量调节
- 🔄 **播放模式** - 支持随机播放和循环播放（单曲/全部）
- 📋 **播放列表** - 管理播放队列
- 📥 **音乐下载** - 一键下载喜欢的歌曲
- 🎨 **现代化 UI** - Material Design 风格，渐变配色
- 📱 **响应式设计** - 支持桌面和移动设备

## 技术栈

- **前端**: HTML5 + Bootstrap 5 + JavaScript (原生)
- **后端**: Python 3 (http.server)
- **图标**: Bootstrap Icons
- **设计**: Material Design 风格

## 快速开始

### 方法 1: 使用启动脚本（推荐）

```bash
cd /home/ihxnan/Music/player
chmod +x start.sh
./start.sh
```

### 方法 2: 直接运行 Python 脚本

```bash
cd /home/ihxnan/Music/player
python3 server.py
```

### 方法 3: 使用 Python HTTP 服务器（简单模式）

```bash
cd /home/ihxnan/Music
python3 -m http.server 8000
```

然后访问: http://localhost:8000/player/

## 访问播放器

启动服务器后，在浏览器中打开:

```
http://localhost:8000
```

## 使用说明

### 播放音乐

1. 在左侧选择艺术家或点击"全部音乐"
2. 在音乐列表中点击任意歌曲开始播放
3. 使用底部播放器控制播放、暂停、切换歌曲

### 搜索音乐

1. 在左侧搜索框中输入歌曲名或艺术家名
2. 从搜索结果中选择要播放的歌曲

### 下载音乐

1. 在音乐列表中找到要下载的歌曲
2. 点击下载按钮（↓图标）
3. 文件将自动下载到您的下载文件夹

### 播放列表

1. 点击右下角的播放列表按钮
2. 查看当前播放队列
3. 点击歌曲可跳转播放
4. 点击 × 可从播放列表中移除

### 播放模式

- **随机播放**: 点击随机按钮（🔀）启用/禁用
- **循环播放**: 点击循环按钮（🔁）切换模式
  - 不循环: 🔁
  - 循环全部: 🔁（高亮）
  - 循环单曲: 🔂

## 注意事项

1. 音乐文件需要放在 `/home/ihxnan/Music/` 目录下的艺术家文件夹中
2. 支持的音频格式: MP3、OGG、FLAC、M4A、WAV
3. 服务器默认运行在 8000 端口
4. 按 `Ctrl+C` 停止服务器

## 自定义配置

如需修改端口，编辑 `server.py` 文件:

```python

```
